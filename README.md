# 3D Graphing Calculator

In this project almost everything is made from scratch, except that I use [GLEW](https://glew.sourceforge.net/) and [RGFW](https://github.com/ColleagueRiley/RGFW).

Parsing and rendering of TTF (True Type Font) is done from scratch. Data structures used are generic and in a [stb](https://github.com/nothings/stb) style single header, I have a seperate repository for the implementaion [cylibx](https://github.com/FilipConic/cylibx).
